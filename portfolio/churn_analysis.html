Methodology

The project follows a standard machine learning pipeline: data exploration, feature engineering, model selection, and hyperparameter tuning. XGBoost was chosen for its ability to handle structured data and its efficiency in high-dimensional spaces.

Feature Importance

Top features: Contract Type, Monthly Charges, Tenure

Implementation Snippet

model = xgb.XGBClassifier(
    n_estimators=100,
    max_depth=5,

)
model.fit(X_train, y_train)

Conclusions

The model successfully identified key drivers of churn. Customers on month-to-month contracts with high monthly charges were found to be at the highest risk. implementing targeted loyalty programs for these segments could significantly improve retention.
